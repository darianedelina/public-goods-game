Инструкция

Представьте, что вы живете в небольшом городе близ реки. Дамба на вашей реке порядком износилась и теперь нуждается в ремонте. Если вы ее не почините, то скорее всего во время паводков дамбу снесет, и пострадают дома всех жителей. Но вы можете вложиться в общий фонд для ремонта дамбы и избежать катастрофы.

Ремонт стоит {{Constants.threshold}} (размер фонда).

Ваш выигрыш определяется по формуле:

Выигрыш = {{ player.working_endowment }} - Вложения + Множитель * {{ player.external_endowment }}

Он рассчитывается следующим образом: очки, которые вы не потратили, остаются вам. Если жители собрали нужную сумму для ремонта, то дамба точно устоит, дома уцелеют. Если средства не собраны, и дамбу не починили, то дамба рухнет с вероятностью {{Constants.probability_of_loss}}. Состояние домов в городе определяет Множитель из формулы. Если дома остались целы, то Множитель = 1, а если дома разрушило - Множитель = 0.

Ваша роль в игре - {% if player.role == 'rich' %} богатый {% else %} бедный {% endif %} житель, это означает, что вы имеете {{ player.working_endowment }}, которые можете потратить на ремонт дамбы, стоимость вашего дома {{ player.external_endowment }}. При этом {% if player.role == 'rich' %} бедные жители имеют {{ Constants.poor_working_endowment }} для ремонта и дом стоимостью {{ Constants.poor_external_endowment }}. {% else %} богатые жители имеют {{ Constants.rich_working_endowment }} для ремонта и дом стоимостью {{ Constants.rich_external_endowment }}. {% endif %}
