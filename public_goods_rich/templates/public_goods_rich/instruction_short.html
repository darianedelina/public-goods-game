Инструкция

Выигрыш = {{ player.working_endowment }} - Вложения + Множитель * {{ player.external_endowment}}

Он рассчитывается следующим образом: очки, которые вы не потратили, остаются вам. Если жители собрали нужную сумму для ремонта, то дамба точно устоит, дома уцелеют. Если средства не собраны, и дамбу не починили, то дамба рухнет с вероятностью {{probability_of_loss_this_group}}.
Состояние домов в городе определяет Множитель из формулы. Если дома остались целы, то Множитель = 1, а если дома разрушило - Множитель = 0.

Ваша роль в этом раунде - {% if player.role == 'rich' %} богатый {% else %} бедный {% endif %} житель.
Это означает, что вы имеете {{ player.working_endowment }}, которые можете потратить на ремонт дамбы, стоимость вашего дома {{ player.external_endowment }}.
При этом {% if player.role == 'rich' %} бедные жители имеют {{ Constants.poor_working_endowment }} для ремонта и дом стоимостью {{ poor_external_endowment_this_group }}. {% else %} богатые жители имеют {{ Constants.rich_working_endowment }} для ремонта и дом стоимостью {{ rich_external_endowment_this_group }}. {% endif %}
